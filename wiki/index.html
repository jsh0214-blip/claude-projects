소개

이 위키는 무엇인가요?

SC 사업본부 2그룹에서 Claude(AI)를 활용해 구축한 업무 자동화 시스템을 설명합니다. Confluence 문서 작성, Gmail 요약, Slack 메시지 발송, 데이터 쿼리까지 — 반복적인 업무를 Claude가 대신 처리합니다.

대상

누가 읽으면 좋나요?

팀 내 구성원뿐 아니라 이 시스템이 어떻게 동작하는지 궁금한 누구든 읽을 수 있습니다. 개발 지식이 없어도 이해할 수 있게 작성했습니다.

핵심

한 줄 요약

Claude Code를 최상위 폴더에 지정해두면, 아래 5개 자동화 프로젝트가 유기적으로 연결되어 작동합니다.

작동 방식

어떻게 연결되어 있나요?

5개 프로젝트는 독립적으로도 동작하지만, 서로 데이터를 주고받으며 더 정교한 자동화를 만들어냅니다.

1
My Info가 정승혁의 Confluence 문서 154개와 Slack 채팅 83,560건을 분석해 페르소나를 구축합니다.
2
Query가 라이브 DB에서 필요한 데이터(매출, 플레이 지표, AB 테스트 결과)를 추출합니다.
3
Confluence가 분석 결과를 바탕으로 기획 문서를 자동 생성합니다.
4
Slack Bot이 채널 유형에 맞는 말투로 메시지를 자동 발송합니다.
5
Gmail Bot이 수신 메일을 요약해 Slack으로 전달합니다.
💡 전체 프로젝트 구조는 구조도 페이지 → 에서 한눈에 확인할 수 있습니다.
프로젝트 1

Confluence 문서 자동화

Confluence에 직접 문서를 쓰는 대신, 필요한 내용을 말하면 Claude가 정승혁 스타일의 기획 문서를 자동으로 생성합니다.

📄
Confluence
studio-g.atlassian.net · REST API 연동
자동화 완료

무엇을 자동화했나요?

Confluence 페이지 생성, 수정, 구조 잡기를 자동화했습니다. 특히 NC(신규 콘텐츠) 시리즈 문서는 항상 동일한 폴더 구조와 양식으로 생성됩니다.

어떤 문서를 만들 수 있나요?

문서 유형설명
기획안NC 시리즈 기획 문서. 배경 → 목적 → 설계 → 로직 구조로 자동 생성
UI/UX 설계서화면 설계 + 백엔드 로직 정의를 테이블 형식으로 정리
AB 테스트 문서분기 조건, 결과 분석을 포함한 운영 문서 자동 생성
로그/그래픽 요청서개발·디자인팀 요청 문서 자동 작성
💡 핵심: "NC_123 큐레이션 기획안 만들어줘" 한 마디면 Confluence에 정승혁 스타일 그대로 문서가 생성됩니다.
프로젝트 2

Gmail Bot — 메일 자동 요약

매일 수신된 Gmail을 Claude가 한국어로 요약해 Slack으로 전송합니다. 중요한 메일을 놓치지 않습니다.

📧
Gmail Bot
Gmail API (OAuth2) · Claude API 요약
자동 실행 중

어떻게 작동하나요?

1
Windows 스케줄러가 매일 지정된 시간에 자동으로 실행
2
Gmail에서 수신 메일 전체 수집
3
Claude API가 각 메일을 핵심만 추려 한국어로 요약
4
요약본을 지정된 Slack 채널로 자동 전송
💡 핵심: 매일 아침 Slack에서 어제 받은 중요 메일을 한눈에 확인할 수 있습니다.
프로젝트 3

Slack Bot

내부 팀 채널 반말, 캐주얼 "ㅋㅋ 이거 어떻게 생각해?"

분석 목적을 말하면 라이브 DB 구조에 맞는 SQL 쿼리를 자동으로 만들어줍니다. Redash와 SQL Workbench 모두 지원합니다.

🗄️
Query
SQL Workbench · Redash · 라이브 DB
DB 학습 완료

연결된 데이터베이스

Contents DB Offer / Product DB 라이브 DB (실서비스)

어떻게 사용하나요?

요청 예시결과
"이번 주 Special Deal 매출 뽑아줘"Redash용 매출 집계 쿼리 생성
"큐레이션 AB 테스트 결과 비교해줘"대조군·실험군 비교 쿼리 생성
"복귀 유저 플레이 지표 보고 싶어"세그먼트별 지표 추출 쿼리 생성
💡 핵심: 라이브 DB 컬럼명과 테이블 구조를 미리 학습해두었기 때문에, 분석 목적만 말하면 바로 쓸 수 있는 쿼리가 나옵니다.
활용 가이드

실제로 무엇을 할 수 있나요?

지금 당장 Claude에게 요청할 수 있는 것들입니다.

문서 작성

📄 "NC_xxx 기획안 만들어줘"

배경·목적·설계·로직 구조로 Confluence에 기획안을 자동 생성합니다. 정승혁 스타일 그대로.

피드백

👔 "그룹장 입장에서 이 기획 봐줘"

정윤호 그룹장 페르소나로 전환, 현실적인 업무 피드백을 반말로 받아볼 수 있습니다.

메시지

💬 "이 내용 슬랙 메시지로 써줘"

채널 유형을 알려주면 공식/내부 말투에 맞게 자동으로 메시지를 작성합니다.

데이터

🗄️ "AB 테스트 결과 쿼리 뽑아줘"

분석 목적을 말하면 라이브 DB에 맞는 SQL 쿼리를 즉시 생성합니다.

퇴고

✍️ "이 문서 내 스타일로 다듬어줘"

작성한 문서나 메시지를 정승혁 페르소나 기반으로 교정합니다.
